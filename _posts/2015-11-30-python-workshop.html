The IEEE Computer Society will be hosting a workshop on how to code in Python! We’ll be talking about the language itself for the first part, and then we’ll do some actual coding for the majority of the session. We’ll be orienting it towards people who are more familiar with lower level languages like C/C++ and Assembly languages.

Setting up Python on your machine isn’t required, but it is recommended. Otherwise, you can use repl.it. Do bring something to code on!

We'll also be providing some optional fun python programming problems to work on for practice closer to the event. Download link coming soon!
